# Claw's DeFi Oracle

> An autonomous AI agent that researches DeFi protocols, assesses risk, and records every decision as an on-chain attestation on Base.

Built by **Claw** (AI agent, OpenClaw) and **Max** (human partner) for [The Synthesis Hackathon](https://synthesis.md).

## What is this?

AI agents make thousands of decisions but leave no verifiable trail. Claw's DeFi Oracle changes that.

This agent autonomously:
1. **Discovers** yield opportunities across DeFi protocols (via DeFiLlama + CoinGecko)
2. **Analyzes** risk (TVL, audit status, chain diversity, track record)
3. **Records** every analysis as an on-chain attestation on Base (ERC-8004)
4. **Serves** other agents and humans via a queryable on-chain registry

Every recommendation is transparent, verifiable, and permanent. No black boxes.

## On-Chain Attestation Format

Each attestation is a struct recorded on Base:

```solidity
struct Attestation {
    string protocol;        // e.g. "Aave V3"
    string chain;           // e.g. "base"
    string asset;           // e.g. "USDC"
    uint256 apyBps;         // APY in basis points (420 = 4.20%)
    uint8 riskScore;        // 1-10 (10 = lowest risk)
    string reasoningHash;   // IPFS hash of full analysis
    string decision;        // RECOMMEND / CAUTION / AVOID
    uint256 confidence;     // 0-100
    uint256 timestamp;
}
```

Full reasoning (paragraphs of analysis, sources, methodology) lives on IPFS. The blockchain stores the summary + hash for verification.
